# مدل سرمایه‌گذاری

نسخه: ۰.۱

وضعیت: سند زنده

---

# معرفی

لوییس او‌اس طراحی شده است تا پروژه‌ها پیش از درخواست سرمایه قابل توجه، اعتبار و وضوح بیشتری پیدا کنند.

هدف این نیست که در مرحله خیلی زود، سرمایه بزرگ درخواست شود.

هدف این است که ابهام مرحله‌به‌مرحله کاهش پیدا کند تا پروژه بتواند نیاز سرمایه بعدی خود را توجیه کند.

---

# نردبان سرمایه‌گذاری

لوییس او‌اس توسعه پروژه را به سطوح پیوسته تقسیم می‌کند.

## سطح اول — آماده‌سازی

نقطه شروع معمول:

**سه مرحله ۹۶.۳ دلاری**

هدف:

- روشن‌کردن ایده
- تعریف پروژه
- آماده‌سازی برای ارائه

این سطح، حداقل مجموعه معتبر دانش و خروجی لازم برای تصمیم جدی بعدی را ایجاد می‌کند.

---

## سطح دوم — نمونه اولیه

نقطه شروع تقریبی:

**۹۶۳ دلار**

هدف:

- ساخت نمایش تصویری یا عملکردی
- آزمایش روایت پروژه
- اعتبارسنجی مسیرهای اصلی کاربر
- تولید دارایی قوی‌تر برای ارائه

این سطح محصول کامل نیست.

مدرکی است که نشان می‌دهد ایده می‌تواند تجربه شود.

---

## سطح سوم — MVP

نقطه شروع تقریبی:

**۹٬۶۳۰ دلار**

هدف:

- ساخت نخستین محصول قابل استفاده
- پیاده‌سازی قابلیت‌های اصلی
- اجرای پایلوت خصوصی یا محدود
- دریافت بازخورد واقعی
- آزمایش فرضیات فنی و تجاری

بودجه دقیق باید از روی معماری تأییدشده پروژه برآورد شود.

---

## سطح چهارم — رشد

نقطه شروع تقریبی:

**۹۶٬۳۰۰ دلار**

هدف:

- تقویت زیرساخت
- توسعه تیم
- افزایش امنیت و پایداری
- گسترش توزیع
- ساخت سیستم فروش و پشتیبانی
- آماده‌شدن برای سرمایه‌گذاری بزرگ‌تر

این سطح فقط زمانی معنا دارد که شواهد واقعی وجود داشته باشد.

---

# سرمایه‌گذار چه زمانی باید وارد شود؟

سرمایه‌گذار نباید فقط به این دلیل وارد شود که ایده جذاب به نظر می‌رسد.

سرمایه‌گذاری زمانی معتبرتر می‌شود که پروژه دارای این موارد باشد:

- مالکیت روشن
- مسئله تعریف‌شده
- بازار قابل شناسایی
- راه‌حل مستند
- هزینه‌های واقع‌بینانه
- نقشه راه تأییدشده
- شواهد تقاضا
- و مسیر اجرایی قابل قبول

لوییس او‌اس پروژه را برای رسیدن به همین نقطه آماده می‌کند.

---

# منابع سرمایه

پروژه می‌تواند از منابع مختلفی رشد کند:

- سرمایه بنیان‌گذار
- درآمد مشتری
- گرنت
- شریک راهبردی
- سرمایه‌گذار فرشته
- سرمایه خطرپذیر
- توافق تقسیم درآمد
- خرید از طریق مارکت‌پلیس
- یا ترکیبی از این روش‌ها

منبع پیشنهادی به بلوغ و ریسک پروژه وابسته است.

---

# درآمد Lewis OS

لوییس او‌اس می‌تواند از مسیرهای زیر درآمد داشته باشد:

- هزینه ثابت آماده‌سازی
- هزینه نمونه اولیه یا ساخت
- هزینه تحویل اولویت‌دار
- هزینه مدیریت پروژه
- کارمزد مارکت‌پلیس
- لایسنس
- خدمات White Label
- سهم درآمد تأییدشده
- یا سهام با توافق جداگانه

هیچ سهم، مالکیت یا تقسیم درآمدی به‌صورت خودکار فرض نمی‌شود.

هرکدام نیازمند قرارداد کتبی مستقل است.

---

# ساختار شفاف بودجه

برای برخی پروژه‌های آینده، می‌توان از مدل تخصیص شفاف استفاده کرد.

نمونه:

- ۳۰٪ — معماری، مدیریت، مستندسازی و دستمزد استودیو
- ۷۰٪ — هزینه‌های مستقیم اجرا

هزینه‌های مستقیم می‌توانند شامل این موارد باشند:

- اعتبار AI و API
- ابزارهای توسعه
- میزبانی
- طراحی
- پیمانکار
- تست
- خدمات حقوقی
- استقرار
- و آزمایش‌های بازاریابی تأییدشده

این مدل باید با دفتر ثبت پروژه و گزارش مبتنی بر Milestone پشتیبانی شود.

# اصل مدیریت ریسک

سطح سرمایه بعدی فقط زمانی باید باز شود که مرحله قبلی شواهد کافی تولید کرده باشد.

سرمایه باید پس از وضوح وارد شود.

وضوح باید از شواهد بیاید.

شواهد باید از کار تکمیل‌شده به دست آید.

---

# تعهد اصلی

لوییس او‌اس تضمین نمی‌کند که هر پروژه سرمایه جذب کند.

اما پروژه را طوری آماده می‌کند که تصمیم سرمایه‌گذاری با وضوح بیشتر، شواهد قوی‌تر و ریسک قابل اجتناب کمتر انجام شود.
