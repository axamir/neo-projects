# نمای کلی

لوییس او‌اس از یک مسیر استاندارد برای آماده‌سازی پروژه استفاده می‌کند.

تمام پروژه‌ها پیش از ورود به مرحله ساخت، از همین معماری عبور می‌کنند.

هدف اصلی:

کاهش ابهام در هر گام.

# مراحل آینده

پس از اعتبارسنجی، پروژه می‌تواند وارد خدمات پیشرفته لوییس او‌اس شود.

مانند:

- Prototype
- MVP
- Production
- Marketplace
- Growth
- Investment Support

این مراحل در نسخه‌های بعدی Lewis OS تکمیل خواهند شد.
