# معرفی

Lewis OS یک معماری لایه‌ای برای خلق پروژه‌هاست.

هر لایه مسئولیت مشخصی دارد.

هیچ لایه‌ای نباید وظیفه لایه پایین‌تر را تکرار کند.

دانش به سمت بالا رشد می‌کند.

اجرا به سمت پایین حرکت می‌کند.
